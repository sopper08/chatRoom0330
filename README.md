## 使用教學
STEP1: 
```
$npm init
```
STEP2:
```
$npm install express socket.io
```
STEP3:
```
$node index.js
```

## 目前功能
* 多人線上聊天.
* 可輸入ID, 讓其他人辨識是誰發的內容.
